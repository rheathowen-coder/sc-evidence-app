# SC Rules of Evidence App

A clean Next.js starter project for a South Carolina Rules of Evidence reference app.

## What is included
- Mobile-friendly home screen
- Browse by article
- Search by rule number, issue, keyword, or full text
- Rule detail page with contextual back button labels
- Favorites saved in local storage
- Caselaw links section with fallback links to the South Carolina Judicial Branch and Justia
- Web app manifest for Home Screen install on iPhone

## What is not included yet
This starter ships with a small sample dataset only. Replace `data/rules.json` with your full 61-rule South Carolina dataset.

## Local run
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Create a free Vercel account.
2. Import this folder as a new project.
3. Accept the default Next.js settings.
4. Deploy.

## iPhone use
Open the deployed URL in Safari, tap Share, then tap Add to Home Screen.
